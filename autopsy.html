Autopsy

This open-source program is included in The Sleuth Kit and provides an easy GUI to display the forensic analytic reportings from an underlying volume. Once we launch the software the only step we had to take for the analysis to be done, is to upload the image we created (samsung.dd) to a new project. 


Autopsy is a very useful tool for this type of forensics. Autopsy for windows allows us to simply upload an image file and it will do most of the hard work for us. Autopsy will find data on the disk and separate it based on the type of data that is found. For example, as seen in figure 4 all of this information was found and presented to the user in a clean and readable manner. Almost immediately we were able to find the objects and content within the phone.

Figure 4.0: Autopsy findings (Image, MP3, memo, contacts)

The figure shows an image, memo, and contact list. All of this was found and made readable on the disk image by just letting the software run the analysis tool. The information was presented in a clean and presentable manner and found the most useful information in terms of actual user data on the device. In addition to finding the files, you would be able to extract them as a read-only file to preserve the integrity of the data. This software allows us to view .db files which are extremely useful in android forensics. In this instance, we were able to view the contact database. However, in a more realistic scenario text messages and call logs are also kept in a .db file. This would be important information to look at when trying to gather information from a cell phone and autopsy makes that experience simple and clean for an investigator who might not have extensive SQL or database forensic knowledge.
